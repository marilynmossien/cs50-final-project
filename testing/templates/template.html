Instructions

Welcome to Timelines!

With Timelines, you can visualize your project to help you organize it.

First, choose a route:

If you're a writer, choose "novel" to plan your next project.
If you're an event planner, choose "event" to start organizing tasks.

Input boxes on the page will prompt you to add information.
For a novel, first input the name, a brief plot summary, the instigating event, and resolution of the story. Then add the rising action, climax, and falling action. Your inputs will map to the timeline in their proper places.
For an event, input the name, start date (when you're beginning to plan), and end date. Then begin to add tasks. Each task requires a start and end date, name, and actor (person assigned to the task). Tasks will add to the timeline in their proper places and will be logged on the bottom of the screen.
